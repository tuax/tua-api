tua-api

🏗 一款可配置的通用 api 请求函数生成工具

快速上手 →

支持多端

支持 web 端、Node 端和小程序端

支持跨域

默认使用 axios，也支持降级为 jsonp

可配置

可配置接口类型、请求方式、默认参数、必填参数等属性

支持 mock

接口数据支持 mock，方便开发调试

中间件

koa 风格中间件，方便添加各种特技

让我们优雅地调用 api~
